# INA ZONE API AUTOMATION

INA Zone API automation using Mocha Chai. This project is a part of the [INA ZONE API](https://github.com/indahmutiah/ina-zone) project.

## Prerequisites

- Node.js
- NPM

## Installation

```bash
npm install
```

## Usage

```bash
npm run test-inazone
```
